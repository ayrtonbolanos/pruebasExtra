Sobre el proyecto: El proyecto se basa en practicas de la ecuacion de la raiz cuadrada de -1, a partir de la cual se desarrolla un software con administracion de proyectos de Kanban automatizado y protegidos bajo el github actions. El proyecto esta desarrollado con modelo Git Branching Model con branch: master, hotfixes, develop y release. Cada issue contiene su definicion, mockup, diagrama de clases y las Pruebas que se necesitan pasar. Nuestros issues tienen un prototipo base, el cual es: landing page, test, y UI. El front se publica en firebase.com y se implementa el code coverage del proyecto.
